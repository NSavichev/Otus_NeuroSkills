# Таблица Шульте

Кроссплатформенное приложение для тренировки внимания и периферического зрения.

## Что это

Таблица Шульте - это упражнение для развития скорочтения и концентрации внимания. Нужно находить числа от 1 до N по порядку как можно быстрее.

## Технологии

- .NET 9
- .NET MAUI
- SQLite
- Entity Framework Core

## Как запустить

```bash
dotnet run --project SchulteTable.Maui -f net9.0-windows10.0.19041.0
```

## Структура

- `SchulteTable.Core` - основная логика
- `SchulteTable.Data` - работа с базой данных  
- `SchulteTable.Maui` - интерфейс приложения

## Функции

- Разные размеры сетки (4x4 до 8x8)
- Сохранение результатов
- Статистика игр
- Темная/светлая тема
- Вибрация при нажатиях
